MyVault Release Notes

All releases are archived here. Current version: v1.0 (2026-03-24).

2026-04-17 · v1.5

1. Item input & editing

  • Added a brand-search modal to Add / Edit Item with curated brand library filtering, auto category suggestions, and themed icons.
  • Purchase info UI refresh: channel / city now support picker + custom input, city history auto-completes, and price fields gain thousands separators.
  • Photo block moved to the top row with unified thumbnail chips and delete controls.
  • “Additional notes” renamed to “Asset Story Log” with full EN/ZH localization.
  • Tag and space fields are governed by Settings toggles; disabling them clears bound data automatically.

2. Batch Edit

  • Batch views reuse the brand-search modal, show tag / space fields dynamically via toggles, and drop the legacy “batch location” input.
  • BatchEditPayload / Service now center on SpaceNode & TagSet data, auto-writing back to the brand library when saved.
  • Introduced city-history logging plus custom city chips, aligned with the single-item form experience.

3. Brand system

  • Brand library schema expanded so BrandItem / BrandType store English, Chinese, pinyin, and logo metadata with normalized search across the app.
  • CSV import normalizes brand names via library metadata and prioritizes logo matches.
  • Brand manager overhaul: denser rows with Chinese labels, new “My Brands / Disabled brands” tabs, multilingual search, and full metadata propagation for new / suggested brands.

4. Settings & preferences

  • Settings adds a space-management toggle with empty states, tighter layout, and selectors (language / theme / currency) that collapse after choosing.
  • Currency display gains a thousands-separator toggle that applies globally to cards and pickers.

5. Other UX polish

  • ImagePicker and Maintenance views inside Edit Item now open as sheets for parity with Add Item.
  • FileImportView dedupe logic upgraded and automatically backfills brand / category / location corrections during import.

v1.0 · 2026-03-24

  • Asset management: create, edit, and delete full asset records.
  • Classification system: categories, tags, brand library, spaces, and locations.
  • Asset photos: capture via camera or import from the photo library.
  • Maintenance logs: record maintenance, repair, cleaning, and upgrade events.
  • Analytics: total value, average value, daily holding cost, category stats, and more.
  • Purchase calendar: review historical purchases by date.
  • Global search: find items by name, brand, or tags.
  • Bulk operations: batch edit and delete.
  • Data import: Excel / Markdown (CSV template supported).
  • E-commerce link capture: parse major marketplace URLs into structured fields.
  • Photo recognition capture: AI-assisted prefill for item data.
  • Backup & restore: generate .zip archives and restore from them.
  • Data export: export asset datasets for archiving or sharing.
