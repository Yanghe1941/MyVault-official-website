1. 我们收集的信息

1.1 用户主动提供的数据

  • 资产信息：名称、价格、购买日期、购买渠道、城市、描述、状态、保修期限等
  • 分类 / 标签 / 品牌 / 空间 / 位置等结构化信息
  • 维护记录：维护类型、日期、费用、服务商、备注
  • 资产照片与图片
  • 场景 / 清单与偏好设置（主题、语言、货币等）

1.2 设备权限信息

  • 相机权限：用于拍照录入资产图片
  • 相册权限：用于选择照片
  • 面容 / 触控 ID：用于本地解锁验证（如用户启用）
  • 文件访问：用于导入 / 导出、备份 / 恢复文件（包括 iCloud Drive）

1.3 我们不会强制收集

  • 账户信息（当前版本不要求注册）
  • 联系人、短信、通话记录等与资产管理无关的数据

2. 信息使用目的

2.1 用于资产管理功能的本地存储与展示。

2.2 用于生成统计分析与可视化结果。

2.3 用于导入 / 导出、备份 / 恢复的本地文件处理。

2.4 用于提升使用体验（语言、主题、货币等偏好）。

3. 数据存储与处理方式

3.1 资产数据默认保存在用户设备本地（Core Data 与本地文件）。

3.2 备份文件为本地生成的 .zip 文件，由用户选择保存到 iCloud Drive 或其他位置。

3.3 导出文件在本地生成，具体文件位置由系统文件选择器或导出流程决定。

3.4 我们不搭建自有云端服务器存储你的资产数据（以当前版本为准）。

4. 第三方服务与共享

4.1 本应用不向第三方出售或共享用户资产数据。

4.2 备份 / 恢复或导出 / 导入过程可能调用系统文件服务（如 iCloud Drive），该过程受 Apple 服务条款约束。

4.3 照片识别界面标注 “Powered by DeepSeek”。若未来版本使用第三方模型服务进行识别，我们将在功能使用前明确提示并征得用户授权。

4.4 数据分析与埋点

我们在 App 内嵌入 Firebase Analytics，用于了解功能使用情况并改善体验。在法律允许范围内，该服务会自动或在特定操作时上报以下信息：

基础信息（自动采集）

  • App 启动次数、会话时长、崩溃与性能指标；
  • 设备信息（型号、系统版本、区域 / 语言设置）以及匿名化实例 ID；
  • 网络状态（Wi-Fi / 蜂窝等）等用于统计的环境信息。

功能事件（自定义埋点）

  • Tab 切换、打开新增页、完成引导等功能操作；
  • Pro 付费相关行为（如 paywall 展示、购买尝试、购买结果）；
  • 触发 Pro 功能门槛时记录入口来源，仅用于分析升级路径。

用途

  • 统计不同功能的使用频率与路径，优化交互与性能；
  • 分析付费漏斗，改进订阅方案与体验；
  • 排查崩溃与异常，保障 App 稳定。

数据处理与存储

  • 分析数据由 Firebase（Google）存储在其云端服务中，我们不会将事件数据与可识别个人的信息直接关联；
  • 我们不会通过 Firebase Analytics 收集通讯录、照片、文档等个人内容；
  • 不会将分析数据共享给第三方广告平台，仅用于产品内的运营与分析。

用户控制

  • 可在系统设置中关闭“允许 App 请求跟踪”或直接卸载 App，以停止此类统计；
  • 亦可通过支持邮箱（hi@moodex.cc）请求删除特定分析数据，我们会配合 Firebase 完成处理。

5. 数据安全

5.1 数据主要保存在设备本地，用户应妥善保管设备与解锁方式。

5.2 应用提供 Face ID / Touch ID 作为本地解锁防护。

5.3 如用户将备份文件存储到云盘，其安全性由用户与云盘服务提供方共同保障。

6. 数据保留与删除

6.1 用户可在应用内删除单项资产或清空所有数据。

6.2 删除后数据将从本地数据库移除；已导出的文件不受影响，用户需自行清理。

6.3 如设备卸载应用，本地数据会随应用卸载而删除（取决于系统机制与用户设置）。

7. 儿童隐私

7.1 本应用面向普通用户，不面向 14 岁以下儿童收集个人信息。

7.2 如发现儿童信息被误收集，请通过联系方式联系我们处理。

8. 协议更新

8.1 我们可能根据功能变化更新隐私协议。

8.2 重大变更将通过应用内提示或更新日志告知。
