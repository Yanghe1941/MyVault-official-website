隐私政策

最近更新日期：2024年3月

1. 数据收集

MyVault 是一款“本地优先”的应用。我们不运行任何后端服务器，也不通过 App 收集、存储或上传您的任何个人资产数据、照片或文字记录。

2. 数据存储与同步

您的所有数据默认存储在您的 iOS 设备本地。如果您启用了 iCloud 同步，数据将通过 Apple 提供的加密通道在您的私有 iCloud 空间内进行流转，开发者无法访问这些数据。

3. 权限说明

相册权限：仅用于您主动选择并添加物品照片。Face ID / Touch ID：仅用于本地解锁应用，生物识别数据由系统保护。iCloud 访问：用于在您的设备间同步资产数据。

4. 第三方服务

本应用不包含任何第三方广告、追踪器或分析插件。我们不会将您的数据出售给任何第三方。

5. 联系我们

如果您对本政策有任何疑问，请通过 GitHub 仓库或应用内的反馈渠道联系开发者。
