MyVault 使用指南

← 返回首页

1. 初次使用

  1. 下载安装：在 App Store 搜索 “MyVault”，下载并打开。
  2. 权限授权：首次运行将按需申请照片、相机、定位等权限，建议根据使用场景授权。
  3. 快速了解界面：底部包含资产、分析、添加、发现、设置五个 Tab，上方主题跟随设置中的黑金 / 日间 / 夜间切换。

2. 资产管理（“资产”Tab）

2.1 新增资产

  1. 点击底部中间的 “+” 或资产页顶部的 “添加”。
  2. 填写名称、品牌（支持中英拼音）、类别、购买时间、价格、货币、城市、渠道等字段，可补充故事、备注、保修日期、图片。
  3. 按需选择空间 / 标签或启用推荐品牌列表，确认后保存。

2.2 编辑 / 删除

在列表左滑或进入详情后修改，底部提供删除入口；批量模式亦可执行删除。

2.3 全局搜索与筛选

  • 点击顶部放大镜或资产页左上角打开搜索面板。
  • 支持按名称、类别、城市、标签、品牌、时间等维度检索，涵盖中文、英文、拼音首字。
  • 列表顶部可切换状态筛选（在用 / 闲置 / 已售等）与排序（时间、价值、日均成本）。

2.4 批量操作

右上角开启“批量”后可勾选多个资产，批量删除、修改标签/空间或添加场景。

3. 资产分析（“分析”Tab）

  1. 查看资产总数、总价值、平均每日持有成本、品牌 / 分类 / 价值分布。
  2. 通过顶部年份选择器查看历史年度数据。
  3. Pro 用户可解锁更细的饼图、折线图与多维度（城市、标签、渠道等）分析。
  4. 保修到期提醒、近期新增资产会以卡片形式提示。

4. 发现页（“发现”Tab）

展示最新功能、品牌灵感与资产管理技巧。若未来接入模板或资讯，也会在此集中更新。

5. 设置（“设置”Tab）

5.1 常用偏好

  • 语言：系统 / 中文 / 英文。
  • 主题：黑金 / 日间 / 夜间，影响全局 UI。
  • 货币单位：选择常用货币（仅影响显示符号）。
  • 千分位显示：开启后金额自动加逗号分隔。
  • 品牌库 / 标签 / 空间功能：可按需启用，关闭后相关字段会隐藏。

5.2 数据管理

  • 导出：输出 CSV 或备份文件，便于归档。
  • 导入：支持 CSV、电商订单表，系统会匹配品牌与类别，可手动校正。
  • 清空数据：慎用，会永久删除本地数据。
  • iCloud 同步：Pro 用户可开启，同步至个人 iCloud 账户。

5.3 安全与 Pro

设置 Face ID / Touch ID / 密码锁保护资产数据，查看或升级 Pro 功能列表。

6. 场景与空间

  1. 在资产详情或批量模式中，可将资产关联到空间（如“家 / 公司 / 仓库”）和场景（自定义组合）。
  2. 适合跨地点管理或按使用场景归类。
  3. 可在发现页或设置中维护既有空间 / 场景。

7. 小技巧

  • 下拉搜索：主界面下拉即可输入关键词，全局搜索支持拼音首字母。
  • 品牌推荐：输入品牌时自动匹配品牌库，涵盖中英文、别名与 Logo。
  • 保修提醒：录入保修截止日期后，可收到维护 / 保修提醒。
  • 电商导入：导入订单文件可自动解析名称、价格、品牌，减少手动录入。
  • 多货币显示：内部金额以人民币存储，切换仅改变符号，方便多地区展示。
  • 千分位输入：开启后价格输入自动带千分位，阅读更清晰。
  • 快速备份：定期导出 CSV / 备份或开启 iCloud，同步后换机直接导入即可恢复。

8. 常见问题

Q：换手机如何迁移数据？
A：在旧设备导出备份或确认 iCloud 已同步，新设备安装 MyVault 后导入备份或登录同一 iCloud。

Q：为什么某些功能看不到？
A：可能相关开关在设置中被关闭，或该功能属于 Pro 版本，请在设置中查看。

Q：导入时品牌不匹配怎么办？
A：导入预览页支持逐条核对与修改，确认拼写后再保存。
