用户指南

MyVault 是一款本地优先的 iOS 资产与纪念物管理工具。本指南覆盖最常用的操作流程。

1. 快速上手

首次打开 App，按引导页完成初始化。在设置中选择 主题（推荐黑金模式）、语言、货币单位。如需增强隐私，可开启 Face ID / Touch ID 应用锁。

2. 新增物品

点击“+”号录入基本信息。除了品牌、价格、渠道与位置，建议添加实拍照片或发票附件。通过“备注”栏，您可以记录物品背后的情感故事或购买初衷。

3. 整理体系

分类：构建清晰的财务目录。标签：轻量的可组合标记。空间：现实位置树图（房产→房间→柜子），帮您精准定位实体资产。

4. 效率工具

支持全局搜索名称、品牌或标签。通过多选进入批量编辑模式，快速完成资产的归类、移动空间或状态变更（如“已送人”、“折旧”）。

5. 数据安全

数据默认存储于本地。开启 iCloud 同步后，数据仅在您的私有云端流转。建议定期使用“ZIP 导出”功能将包含附件的完整数据备份到其他安全位置。
