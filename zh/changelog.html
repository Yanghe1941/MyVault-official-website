v1.7 2026-04-27

1. 功能优化

  • 资产概览卡片：净资产改为等同「使用中资产价值」，资产数量显示为「总数 / 使用中」，并让净资产文字与总资产左对齐。
  • 资产详情页：日期区间只显示一次，品牌 Logo / 信息卡 / 维护卡在黑金或系统深色模式下自动切换配色，避免白底白字。
  • 资产故事线：改为可展开 / 折叠的时间线列表，每条记录自动带时间戳，空态增加提示文案与右上角「+」快速新增入口。
  • 维护记录卡片：增加「+」按钮快速进入新增维护，空列表时提示「暂无维护记录」。

2. 新增功能

  • 导入失败明细：导入完成后展示缺名称、价格无效、文件内重复、库内已存在等具体原因与行数，帮助快速定位问题。
  • 故事线录入弹窗：新增 StorylineEntryComposer，可在详情页内一次编写并保存多条故事记录。

3. Bug 修复

  • 资产卡 / 网格状态缺失：列表与网格视图的状态徽章恢复显示，并使用 normalizedStatus 兼容旧数据。
  • 资产详情状态 / 编辑页面：状态字段初始化与详情展示逻辑统一，确保编辑页与详情页读取一致。
  • 导入格式冲突：FormatCard 重复定义导致编译报错，已更名为 ImportFormatCard。
  • App Icon 资源：移除 AppIcon 集合内多余的 “xxx 2.png” 文件，解决 “unassigned children” 报错。
  • Brand Logo 白底：在黑金主题下强制白底的品牌 Logo 恢复纯白背景，保持视觉一致性。
v1.6 2026-04-21

1. 云同步与设置

  • 设置页新增独立「云同步」子菜单，集中展示同步状态、账号信息、最近成功/失败记录，并整合备份与恢复入口。
  • 优化云同步开关切换逻辑，减少界面卡顿，并增加超时兜底处理。
  • 设置文案与视觉更新：「资产展示」改为「资产视图」，「卡片/网格模式」同步改名，整体风格与云同步页保持一致。

2. 文件导入体验

  • 重构导入页布局，绿色/蓝色主按钮文字居中，模板下载文案支持中英文切换。
  • 导入记录支持分页浏览，导入完成后可直接返回首页查看结果。
  • 增强购买时间解析，兼容更多日期格式，导入更稳健。

3. 资产视图与列表

  • 新增卡片视图、网格视图、极简视图三种模式，并自动记忆用户选择。
  • 卡片与网格视图的状态图标统一表现，「使用中」状态加入轻微呼吸动画，极简视图进一步压缩间距。

4. 分析与曲线

  • 资产曲线改为单曲线筛选模式，配合调整后的分析卡片顺序与状态展示。
  • 曲线渲染与状态提示优化，读数更直观。

5. 资产编辑

  • 补齐缺失的「状态」字段，编辑页标题统一为「编辑资产」。

6. 稳定性与本地化

  • 增强发布稳定性：降低启动崩溃风险、优化持久化存储、提升导入与云同步流程健壮性。
  • 中英文本地化内容更完整，云同步与设置页视觉风格统一。

7. 使用说明

  • 云同步仍依赖设备 iCloud / CloudKit 环境；极端日期格式可能仍需手动调整。
v1.5 2026-04-17

1. 资产录入与编辑

  • 新增品牌搜索弹窗，可从品牌库筛选并自动推荐分类，沿用主题化图标。
  • 购买信息 UI 改造：渠道 / 城市支持菜单 + 自定义输入，城市历史自动补全，价格字段带千分位格式化。
  • 图片信息区上移到表单首行，统一缩略图组件并支持删除。
  • “补充说明”更名为“资产故事记录”，中英文本地化同步。
  • 标签、空间字段受设置开关控制，禁用后自动清空关联数据。

2. Batch Edit 批量修改

  • 批量页复用品牌搜索弹窗，标签 / 空间按开关动态显示，移除旧的“批量设置位置”。
  • BatchEditPayload / Service 以 SpaceNode & TagSet 为核心，保存时自动写入品牌库。
  • 新增城市历史写入与自定义城市 chips，体验与录入页一致。

3. 品牌体系

  • 品牌库扩展：BrandItem / BrandType 包含中英 / 拼音 / logo，并提供统一归一化检索。
  • 导入 CSV 时根据品牌元数据规范名称并优先匹配 logo。
  • 品牌管理页重构：列表更紧凑、显示中文名，新增“我的品牌库 / 未启用品牌”分段，多语言搜索结果同步；新增 / 推荐品牌自动带全量元数据。

4. 设置与偏好

  • Settings 增加空间管理开关与空态提示，偏好布局更紧凑，语言 / 主题 / 货币选择器选中后自动收起。
  • 货币显示新增千分位开关，并在资产卡片与选择器等处统一生效。

5. 其他体验改进

  • ImagePicker、Maintenance 视图在 EditItem 中改为 sheet 呈现，与 Add/Edit 交互一致。
  • FileImportView 去重逻辑升级，导入时自动补全品牌库、类别与位置的重复清理。
v1.0 2026-03-24
  • 资产管理：支持资产记录、编辑与删除。
  • 分类体系：类别、标签、品牌库、位置与空间管理。
  • 资产照片：拍照或从相册选择照片。

  • 全局搜索：按名称、品牌、标签进行快速查找。
  • 批量管理：支持批量编辑与删除。
  • 数据导入：支持 Excel/Markdown 导入（CSV 模板）。
  • 电商链接录入：识别主流电商链接并预填信息。
  • 照片识别录入：AI 辅助识别并预填字段。
  • 备份与恢复：生成 .zip 备份并支持恢复。
  • 数据导出：可导出资产数据记录。
