MyVault 用户协议

生效日期：2026-03-24 · 更新日期：2026-03-24 · 适用产品：MyVault（“本应用”）

← 返回首页

1. 定义

1.1 “我们 / 本应用 / 服务提供方”指 MyVault 的开发与运营主体（杨鹤）。

1.2 “用户”指下载、安装或使用本应用的自然人。

1.3 “内容 / 数据”指用户在本应用中录入或导入的资产信息、照片、标签、分类、空间、位置、维护记录、偏好设置等。

2. 服务说明

2.1 本应用为个人现实资产管理工具，支持资产记录、照片管理、分类与标签管理、空间 / 位置管理、维护记录、数据导入导出、备份与恢复、资产统计分析等功能。

2.2 本应用采用本地数据存储为默认模式，不要求用户注册账户即可使用（以当前版本为准）。

2.3 备份 / 恢复通过用户手动选择文件完成，本应用不会在未获授权的情况下向第三方服务器上传数据。

3. 许可与使用

3.1 我们授予用户一项非排他、不可转让、可撤销的许可，用于安装并在个人设备上使用本应用。

3.2 用户仅可为个人合法用途使用本应用，不得用于非法活动或侵犯他人合法权益的行为。

3.3 用户不得对本应用进行反向工程、反编译、恶意修改或以任何方式破坏安全机制。

4. 用户数据与责任

4.1 用户对其输入、导入、编辑的内容与数据真实性、合法性负责。

4.2 由于用户误操作、设备损坏、系统升级或未及时备份造成的数据丢失，由用户自行承担风险。

4.3 本应用提供的统计分析与提示仅供参考，不构成任何形式的财务或法律建议。

5. 隐私与安全

5.1 我们尊重并保护用户隐私，详细规则见《MyVault 隐私协议》。

5.2 用户应妥善管理设备解锁、Face ID / Touch ID 或密码，以免他人未经授权访问资产信息。

6. 功能变更与服务中断

6.1 我们可根据产品规划对功能进行优化、调整或新增。

6.2 因系统维护、设备故障或不可抗力导致的短期中断，我们将尽力恢复但不承担由此造成的直接或间接损失。

7. 免责声明与责任限制

7.1 在法律允许的范围内，本应用按“现状”提供，不对适销性、特定用途适用性作任何明示或默示保证。

7.2 对于使用本应用导致的任何间接损失、数据丢失或经济损失，我们不承担责任。

8. 适用法律与争议解决

8.1 本协议受中华人民共和国法律管辖（如面向境外用户发布，请调整）。

8.2 因本协议产生的争议，双方应协商解决；协商不成的，提交服务提供方所在地有管辖权的人民法院解决。
